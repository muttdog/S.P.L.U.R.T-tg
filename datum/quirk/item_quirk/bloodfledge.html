on_enter_coffin
(/mob/living/carbon/target, /obj/structure/closet/crate/coffin/coffin, /mob/living/carbon/user)

Coffin check for Bloodfledges. Enables quirk processing if all conditions pass.

Requires the following

  • Organic mob biotype
  • No HOLY anti-magic
  • No garlic reagent
  • No stake embedded

Blood update signal handler for Bloodfledges

  • Warns normally if the target is conscious
